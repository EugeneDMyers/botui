botui
=====

Botui is a device-independent graphical interface designed initially for the Kovan controller.
It also serves as the base of the Kovan simulator for both Mac OS X, Windows, and Linux.

Requirements
============

* easydevice
* tinyarchive
* zlib
* kiss-compiler
* CMake 2.6.0 or later
* Qt 4.7.4 or later
